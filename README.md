# SXSW HOT or NOT

**Insider leaderboard for SXSW 2026.** Swipe right if it's fire. Left if it's mid. See what the crowd actually thinks.

**[sxswhotornot.com](https://sxswhotornot.com)**

## How it works

1. Enter your name
2. Swipe through 12 hot topics — American Dynamism, Peptides, Crypto, AI Agents, and more
3. See live vote results after each swipe
4. Climb the leaderboard

## Stack

- **Next.js 15** — App Router, React 19, Edge Runtime
- **Framer Motion** — Swipe gestures and animations
- **Tailwind CSS** — Styling
- **Vercel** — Hosting and serverless functions
- **Telegram Web App** — Bot integration via [@SWSW_HotOrNot_Bot](https://t.me/SWSW_HotOrNot_Bot)

## Running locally

```bash
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000).

## Environment variables

| Variable | Description |
|----------|-------------|
| `TG_BOT_TOKEN` | Telegram bot token from @BotFather |
| `NEXT_PUBLIC_APP_URL` | Deployed app URL |
